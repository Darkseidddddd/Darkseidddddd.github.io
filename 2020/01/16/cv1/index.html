应用于Potsdam数据集的实时语义分割模型的研究
发表于 2020-01-16| 更新于 2020-02-11|机器学习语义分割
  • 项目构思来历
    • 遥感图像语义分割是遥感图像应用的核心技术之一，通过像素级的语义分割，能对遥感图像所包含的物体类别及分布有全局的了解，初步实现遥感图像的自动理解，并为后续的目标检测、高层语义提取等任务提供基础。近年来，深度学习在计算机视觉中的图像分类、图像语义分割、目标检测等领域得到了广泛应用并取得了很好的效果。
    • 在分割问题中很难直接使用CNN网络。CNN的识别是图像级的识别，在卷积层之后需要使用全连接层得到固定长度的特征向量，而且CNN的池化层会丢弃部分位置信息。我们根据BiSeNet模型结合其它类型的神经网络，进行遥感图像的语义分割，可以实现网络的加速、在不增加太多计算量的同时增大感受野、较好的保留边缘信息。
  • 目的、意义等
    • 将BiSeNet等模型引入遥感图像语义分割，结合其它语义分割框架的有点，进一步提高算法精度，实现基于深度学习的遥感图像语义分割。
    • 大多数传统模式识别方法采用手工设计的特征，它主要依靠设计者的先验知识，允许出现的参数较少。深度学习能从训练数据中自动提取多层次的图像特征，从而克服了传统方法需要专业知识才能更好地设计特征的难点。
    • 遥感图像的语义分割在土地利用分类、环境监测、道路检测、国土资源管理、精细农业、智慧城市、防灾救灾等应用领域上都有重要意义。

实验一

一. 数据集

Potsdam RGB 三通道

总共六分类

surfaces (RGB: 255, 255, 255)

Building (RGB: 0, 0, 255)

Low vegetation (RGB: 0, 255, 255)

Tree (RGB: 0, 255, 0)

Car (RGB: 255, 255, 0)

Clutter / background (RGB: 255, 0, 0)

训练集总共24张6000 × 6000 × 3的图片，测试集总共14张6000 × 6000 × 3的图片

把所有训练集和测试集切割为512 × 512 × 3的图片，对于每行每列最后一张除不尽的部分，用上一张图片最后的部分像素填充至512 × 512大小。

切割后的训练集有3456张图片，测试集有2016张图片，大小为512 × 512 × 3。

这里把训练集，测试集按6 : 2 : 2分成训练集、验证集和测试集，此时训练集有3288张图片，验证集有1092张图篇，测试集有1092张图片。

二. 实验参数

backbone：resnet18

1080 8G × 1

epochs：1000

batch_size：2

初始lr：0.025

optimizer：sgd

训练时长：99h

其中init_lr为初始学习率，iter为迭代次数，max_iter为总的训练迭代次数，power为指数，这里设为0.9。
